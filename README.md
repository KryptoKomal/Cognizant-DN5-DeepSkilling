# Week 1 - Engineering Concepts

## Overview

This repository contains the hands-on exercises and implementations completed as part of **Week 1 – Engineering Concepts** of the Cognizant DN 5.0 Deep Skilling Program.

The objective of this week is to build a strong foundation in software engineering concepts through Design Patterns, SOLID Principles, and Data Structures & Algorithms.

---

## Topics Covered

### 1. Design Patterns and Principles

The following design principles and patterns are implemented:

#### SOLID Principles

* Single Responsibility Principle (SRP)
* Open/Closed Principle (OCP)
* Liskov Substitution Principle (LSP)
* Interface Segregation Principle (ISP)
* Dependency Inversion Principle (DIP)

#### Design Patterns

* Singleton Pattern
* Factory Method Pattern
* Builder Pattern
* Adapter Pattern
* Decorator Pattern
* Proxy Pattern
* Observer Pattern
* Strategy Pattern
* Command Pattern
* MVC Pattern
* Dependency Injection

### 2. Data Structures and Algorithms

The following concepts and exercises are covered:

* Algorithm Analysis
* Time Complexity
* Space Complexity
* Big O Notation
* Arrays
* Linked Lists
* Linear Search
* Binary Search
* Bubble Sort
* Quick Sort
* Recursive Algorithms

## Hands-on Exercises

### Exercise 1 – Configuring a Basic Spring Application

- Created a Maven project
- Configured `applicationContext.xml`
- Defined `BookService` and `BookRepository` beans
- Used Spring IoC Container

### Exercise 2 – Implementing Dependency Injection

- Implemented Setter Dependency Injection
- Wired `BookRepository` into `BookService`
- Verified dependency injection using Spring Container

### Exercise 4 – Creating and Configuring a Maven Project

- Created Maven project structure
- Added Spring Context, Spring AOP and Spring WebMVC dependencies
- Configured Maven Compiler Plugin for Java 1.8

### Spring Data JPA – Quick Example

- Created Spring Boot project using Spring Initializr
- Configured MySQL database
- Created Entity, Repository, Service and Main class
- Retrieved records using Spring Data JPA

### Difference between JPA, Hibernate and Spring Data JPA

- JPA is a Specification
- Hibernate is an ORM Framework implementing JPA
- Spring Data JPA simplifies Hibernate by reducing boilerplate code
